How to use this Weather App ?

In the Home Page the Website already shows the weather of Bangalore as default,
As you may see on the top right corner of Home page there's a search bar, you can check weather
for your favorite city and get all the details of that particular city.

What is the component used to show weather data ?

RapidAPI - We've used an API to make the main part of this project work and run smooth
It is an API which gives 96% of accuracy While showing the realtime weather information

Where is the Login or Sign-Up data stored ?

It is stored in a database, the collected data is actually just used for the validation
purpose and is stored on XAMPP SERVER to give personalised weather Information.
